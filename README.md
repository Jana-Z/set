# An implementation of SET
Play set online

## Setup
Clone repo and run `npm install` to get all relevant dependencies.  
Run `npm start` to start the app locally.

## Technologies
- [React](https://reactjs.org/)
- [React Bootstrap](https://react-bootstrap.github.io)
